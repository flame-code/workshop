73 ! double precision x(n),fvec(m),fjac(ldfjac,n)

75 ! if iflag = 1 calculate the functions at x and

78 ! return this matrix in fjac. do not alter fvec.
79 ! ----------
80 ! return
81 ! end

88 ! of functions.
89 !
90 ! n is a positive integer input variable set to the number
91 ! of variables. n must not exceed m.
92 !
93 ! x is an array of length n. on input x must contain
94 ! an initial estimate of the solution vector. on output x
95 ! contains the final estimate of the solution vector.
96 !
97 ! fvec is an output array of length m which contains
98 ! the functions evaluated at the output x.
99 !
100 ! fjac is an output m by n array. the upper n by n submatrix
101 ! of fjac contains an upper triangular matrix r with
102 ! diagonal elements of nonincreasing magnitude such that
103 !
104 ! t t t
105 ! p *(jac *jac)*p = r *r,
106 !
107 ! where p is a permutation matrix and jac is the final
108 ! calculated jacobian. column j of p is column ipvt(j)
109 ! (see below) of the identity matrix. the lower trapezoidal
110 ! part of fjac contains information generated during
111 ! the computation of r.
112 !
113 ! ldfjac is a positive integer input variable not less than m

115 !
116 ! ftol is a nonnegative input variable. termination
117 ! occurs when both the actual and predicted relative
118 ! reductions in the sum of squares are at most ftol.
119 ! therefore, ftol measures the relative error desired
120 ! in the sum of squares.
121 !
122 ! xtol is a nonnegative input variable. termination
123 ! occurs when the relative error between two consecutive
124 ! iterates is at most xtol. therefore, xtol measures the
125 ! relative error desired in the approximate solution.
126 !
127 ! gtol is a nonnegative input variable. termination
128 ! occurs when the cosine of the angle between fvec and
129 ! any column of the jacobian is at most gtol in absolute
130 ! value. therefore, gtol measures the orthogonality
131 ! desired between the function vector and the columns
132 ! of the jacobian.
133 !
134 ! maxfev is a positive integer input variable. termination
135 ! occurs when the number of calls to fcn with iflag = 1

137 !
138 ! diag is an array of length n. if mode = 1 (see

140 ! must contain positive entries that serve as
141 ! multiplicative scale factors for the variables.
142 !
143 ! mode is an integer input variable. if mode = 1, the

145 ! the scaling is specified by the input diag. other
